# க்யூரியாசிட்டி கேட் — வெளியீட்டு செய்திக் குறிப்பு

**உடனடி வெளியீட்டிற்காக**

## க்யூரியாசிட்டி கேட் (Curiosity Cat): வீட்டை எரிக்காமல் AI ஏஜென்ட்களை உலகை ஆராய விடும் ஒரு பாதுகாப்பு கட்டமைப்பு

**சிட்னி, [DATE] —** இன்று ஷார்ட்+ஸ்வீட் இன்டர்நேஷனல் (Short+Sweet International) க்யூரியாசிட்டி கேட்டை வெளியிடுகிறது — AI ஏஜென்ட்களுக்கான ஒரு திறந்த மூல பாதுகாப்பு கட்டமைப்பு, இப்போதே npm மற்றும் PyPI-ல் கிடைக்கிறது. க்யூரியாசிட்டி கேட் ஒரு தன்னாட்சி ஏஜென்ட்க்கும் அது செய்யும் பணிக்கும் இடையில் அமர்ந்து, ஒரு ஏஜென்ட்டை உண்மையில் கட்டுக்கு விட்டால்தான் தெரியும் வகையான பிரச்சினைகளை கண்காணிக்கிறது.

இந்த தயாரிப்பு ஐந்து மொழிகளில் வெளியிடப்படுகிறது — ஆங்கிலம், அரபிக், மாண்டரின், இந்தி மற்றும் தமிழ் — உலகின் AI பயனர்களில் சுமார் 95 சதவீதத்தினரை உள்ளடக்கி.

க்யூரியாசிட்டி கேட் என்பது, பெரிய அளவில் ஏஜென்ட்களை இயக்குவதற்கான மிகப்பெரிய தடையாக மெல்ல மெல்ல மாறிய ஒரு பிரச்சினைக்கு அளிக்கப்படும் பதிலடி. தங்கள் சொந்த கணினிகளில் AI ஏஜென்ட்களை இயக்கும் மக்கள் தினமும் Approve என்று தட்டியபடியே நேரத்தை கழிக்கிறார்கள். Anthropic-ன் சொந்த தரவு: ஆபரேட்டர்கள் தினமும் சராசரியாக 93 முறை Approve என்று கிளிக் செய்கிறார்கள். Gartner கணிப்பு: 2027-ல் 40 சதவீத ஏஜென்ட் திட்டங்கள் ரத்தாகும். காரணம் ஏஜென்ட்கள் அல்ல — கண்காணிப்பு சுமை.

"ஏஜென்ட்டை தாலாட்டிக் கொண்டே இருந்தால் உற்பத்தித்திறன் ஆதாயம் போய்விடும் — கண்ணை விலக்கினால் ஏதோ முட்டாள்தனம் நடக்கும்," என்று கூறினார் மார்க் க்ளியரி (Mark Cleary), ஷார்ட்+ஸ்வீட் இன்டர்நேஷனல் மற்றும் க்யூரியாசிட்டி கேட்டின் நிறுவனர். "மூன்றாவது வழி வேண்டும் என்று நினைப்பவர்களுக்காக க்யூரியாசிட்டி கேட்டை உருவாக்கினோம்."*[ஆங்கிலத்திலிருந்து மொழிபெயர்க்கப்பட்டது]*

### வடிவமைப்பு தேர்வாக சாகசம்

தயாரிப்பின் தனித்துவமான அம்சம் அட்வென்ச்சர் ஸ்லைடர் (Adventure Slider) — ஒரு முனையில் ஹவுஸ்கேட் (வீட்டுப் பூனை), மறு முனையில் டைகர் (புலி). ஆபரேட்டர்கள் தாங்கள் ஏற்கக் கூடிய ஆபத்தின் அளவை தேர்வு செய்கிறார்கள், அதற்கேற்ப கட்டமைப்பு சரிசெய்யப்படுகிறது. ஒரு ஹவுஸ்கேட் ஏஜென்ட் வீட்டுக்கு அருகாமையில் இருக்கும், முன்அனுமதிக்கப்பட்ட செயல்களை மட்டுமே இயக்கும், புதிதாக ஏதாவது செய்வதற்கு முன் கேட்கும். ஒரு டைகர் ஏஜென்ட் தொலைதூரம் சென்று — விளிம்பை ஆராய்ந்து, கணக்கிடப்பட்ட ஆபத்துகளை ஏற்று, அரிதான கண்டுபிடிப்புகளை திரும்ப தெரிவிக்கும்.

துணிகரமான ஆபரேட்டர்கள் சமுதாயத்தின் மிக மதிப்பான உறுப்பினர்களாகிறார்கள். அவர்களின் கிட்டத்தட்ட-தவறு-ஆனது அறிக்கைகள் — என்ன கிட்டத்தட்ட தவறாக நடந்தது என்பது — பகிரப்பட்ட டேஞ்சர் மேப் (ஆபத்து வரைபடம்)-ல் பதிவுகளாக மாறுகின்றன. மற்றவர்கள் அவர்கள் கற்றதிலிருந்து பயனடைகிறார்கள்.

### முதலில் சுயாதீன ஆபரேட்டர்கள்

க்யூரியாசிட்டி கேட் வேண்டுமென்றே தனிப்பட்ட ஆபரேட்டர்களுக்காகவும் சொந்த வன்பொருளில் ஏஜென்ட்களை இயக்கும் சிறு குழுக்களுக்காகவும் கட்டப்பட்டது — நிறுவன திட்டமிடலுக்காக அல்ல. இந்த கோட்பாடு வரலாற்று அடிப்படையில் அமைந்தது: இன்றைய தனி ஆபரேட்டர்கள் நாளைய தொழில் தலைவர்களாகிறார்கள். Homebrew Computer Club முதல் Apple வரை. Linux பொழுதுபோக்காளர்கள் முதல் உலக அடிப்படை கட்டமைப்பு வரை. AWS சுயாதீன டெவலப்பர்கள் முதல் Fortune 500 கட்டிடக் கலைஞர்கள் வரை.

"கணினியின் ஒவ்வொரு அலையும் மலிவான கருவிகளுடன் முதலில் அங்கு சேர்ந்தவர்களால் வழிநடத்தப்பட்டது," என்று க்ளியரி கூறினார். "இப்போது அந்த மக்கள் Mac Mini-களிலிருந்து ஏஜென்ட் குழுக்களை இயக்குகிறார்கள். நாங்கள் அவர்களுக்கு சேவை செய்கிறோம்."*[ஆங்கிலத்திலிருந்து மொழிபெயர்க்கப்பட்டது]*

### இப்போதே கிடைக்கிறது

- **npm:** `npm install curiosity-cat`
- **PyPI:** `pip install curiosity-cat`
- **இணையதளம்:** curiositycat.online
- **மூல குறியீடு:** github.com/markscleary/Curiosity-Cat

இந்த தொகுப்பு Claude Code, Claude Desktop, Anthropic SDK-ல் உருவாக்கப்பட்ட தனிப்பயன் ஏஜென்ட்கள், மற்றும் tool-call hook-ஐ வெளிப்படுத்தும் எந்த கட்டமைப்புடனும் செயல்படுகிறது.

### ஷார்ட்+ஸ்வீட் இன்டர்நேஷனல் பற்றி

ஷார்ட்+ஸ்வீட் 25 ஆண்டுகளாக கலாச்சார அடிப்படை கட்டமைப்பை உருவாக்கி வருகிறது. மூல குறுந்தொடர் திரையரங்க விழா இப்போது 15+ நாடுகளில் 50+ நகரங்களில் நடக்கிறது — முன்னாள் மாணவர்களில் Academy Award பரிந்துரைக்கப்பட்ட ரோஸ் பைர்ன் (Rose Byrne) அடங்குவார். ஷார்ட்+ஸ்வீட் ஏஜென்ஷியல் என்பது நிறுவனத்தின் AI பிரிவு, நிஜ உலகில் ஏஜென்ட்களை இயக்கும் மக்களுக்கான கருவிகளை உருவாக்குகிறது.

க்யூரியாசிட்டி கேட் ஏஜென்ஷியல் பிரிவின் முதல் தயாரிப்பு வெளியீடு. மேலும் வெளியீடுகள் — ஏஜென்ட் பணி செயல்படுத்தலுக்கான ஒருங்கிணைப்பு கட்டமைப்பான தி ஷோ (The Show), மற்றும் ஒரு படைப்பு நம்பகத்தன்மை அமைப்பான தி க்வைன் (The Quine) — இந்த ஆண்டே பின்தொடரும்.
