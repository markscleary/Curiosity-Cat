# க்யூரியாசிட்டி கேட் — நிறுவனர் அறிக்கை

Approve என்று தட்டுவதில் சலித்ததால் க்யூரியாசிட்டி கேட்டை கட்டினேன்.

சிட்னிக்கு தெற்கே இரண்டு மணி நேர தூரத்தில் உள்ள திரூல் (Thirroul)-ல் என் வீட்டு அலுவலகத்தில் Mac Mini-யில் AI ஏஜென்ட்களை இயக்குகிறேன். ஆறு ஏஜென்ட்கள், ஷார்ட்+ஸ்வீட் இன்டர்நேஷனல் முழுவதும் வெவ்வேறு வேலைகள் செய்கின்றன — தொடர்பாடல் ஆராய்ச்சி, ஒப்பந்த மதிப்பாய்வு, விளம்பர வரைவு, படம் மேலாண்மை, ஏஜென்ட் தகவல்தொடர்பு, அடிப்படை கட்டமைப்பு பணி. அவை பயனுள்ளவை. என் நேரத்தை மிச்சப்படுத்துகின்றன. அவற்றில் ஒன்றையாவது திரையில் நான் இல்லாமல் வேலை செய்ய விட முயற்சித்த தருணம், முழு ஏற்பாடும் தகர்ந்துவிட்டது.

பிரச்சினை ஏஜென்ட்கள் அல்ல. பிரச்சினை கண்காணிப்பு சுமை. அவர்கள் மேற்கொள்ள விரும்பும் ஒவ்வொரு செயலும் ஒரு கோரிக்கையை உருவாக்குகிறது. ஒவ்வொரு கோரிக்கையும் அவர்கள் என்ன செய்யப்போகிறார்கள் என்பதை படிக்க, விளைவுகளை புரிந்துகொள்ள, Approve அல்லது Cancel என்று கிளிக் செய்ய வேண்டுகிறது. Anthropic தரவை வெளியிட்டது: ஆபரேட்டர்கள் தினமும் சராசரியாக 93 முறை Approve என்று கிளிக் செய்கிறார்கள். என்னுடையது அதிகம். சரிபார்த்தேன்.

அதிலிருந்து இரண்டு வழிகள். முதலாவதாக, அனுமதித்துக்கொண்டே இருக்கலாம் — அதாவது உங்கள் வேலையை ஏஜென்ட் செய்யும் வேலையை கண்காணிக்கும் வேலையால் மாற்றிவிட்டீர்கள், இது நோக்கத்திற்கு நேர்மாறானது. இரண்டாவதாக, கண்ணை விலக்கலாம் — இதுதான் பெரும்பாலான மக்கள் செய்வது, மேலும் AI ஏஜென்ட்கள் கட்டுக்கு மீறி நடந்ததாக செய்திகள் வருவதற்கும் இதுவே காரணம்.

மூன்றாவது வழி வேண்டும் என்று விரும்பினேன். க்யூரியாசிட்டி கேட் அதுதான்.

தனித்துவமான அம்சம் அட்வென்ச்சர் ஸ்லைடர். ஒரு முனையில் ஹவுஸ்கேட் (வீட்டுப் பூனை), மறு முனையில் டைகர் (புலி). ஆபரேட்டர்கள் தாங்கள் ஏற்கக் கூடிய ஆபத்தின் அளவை தேர்வு செய்கிறார்கள், அதற்கேற்ப கட்டமைப்பு சரிசெய்யப்படுகிறது. ஒரு ஹவுஸ்கேட் ஏஜென்ட் வீட்டுக்கு அருகில் இருக்கும், முன்அனுமதிக்கப்பட்ட செயல்களை மட்டுமே இயக்கும், புதிதாக ஏதாவது செய்வதற்கு முன் கேட்கும். ஒரு டைகர் ஏஜென்ட் தொலைதூரம் செல்கிறது, கணக்கிடப்பட்ட ஆபத்துகளை ஏற்கிறது, அங்கு என்ன கண்டது என்று திரும்ப தெரிவிக்கிறது.

மிகவும் துணிகரமான ஆபரேட்டர்கள் — டைகரில் ஏஜென்ட்களை இயக்குபவர்கள் — வேறு யாரும் இன்னும் பார்க்காதவற்றை சந்திக்கிறவர்கள். அலைந்து தூரம் சென்ற ஏஜென்ட். யாரும் எதிர்பாராத ஒன்று செய்த கருவி. ஒரு பொறியாக மாறிய இணையதளம். அந்த ஆபரேட்டர்கள் கிட்டத்தட்ட-தவறு-ஆனது அறிக்கைகளை எழுதுகிறார்கள் — என்ன கிட்டத்தட்ட தவறாக நடந்தது — அந்த அறிக்கைகள் டேஞ்சர் மேப்-ல் பதிவுகளாகின்றன. மற்றவர்கள் அவர்கள் கற்றதிலிருந்து பயனடைகிறார்கள்.

இந்த மாதிரி விமானப் போக்குவரத்திலிருந்து கடன் வாங்கப்பட்டது. விமானிகள் ASRS அறிக்கைகளை தாக்கல் செய்கிறார்கள். அந்த அறிக்கைகள் பகிரப்பட்ட தரவுத்தளத்திற்கு செல்கின்றன. எதிர்கால விமானிகள் கடந்த கால தவறுகளிலிருந்து கற்றுக்கொள்கிறார்கள். அறிக்கை அனாமதேயமானது. துணிகரமான ஆபரேட்டர்கள் தங்களை கிட்டத்தட்ட கொன்றதை பற்றி உண்மையை கூறுவதால் அமைப்பு பாதுகாப்பாகிறது.

அதுதான் தயாரிப்பு. நான் ஒரு கணம் பேச விரும்புவது கோட்பாட்டை பற்றி.

25 ஆண்டுகளாக ஒரு கலை வணிகத்தை நடத்தி வருகிறேன், சுயாதீன ஆபரேட்டர்களின் ஒவ்வொரு அலையும் அடுத்த அலையின் தொழில் தலைவர்களாகும் வடிவம் மிகவும் தொடர்ச்சியாக இருக்கிறது, அது வடிவமாக இல்லாமல் விதியாக மாறிவிட்டது. Homebrew Computer Club Apple ஆனது. Linux பொழுதுபோக்காளர்கள் உலக அடிப்படை கட்டமைப்பு ஆனார்கள். AWS டெவலப்பர்கள் Fortune 500 கட்டிடக் கலைஞர்கள் ஆனார்கள். இந்த வடிவம் நிலைக்கிறது ஏனென்றால் மலிவான கருவிகளுடன் முதலில் அங்கு சேர்ந்தவர்கள்தான் உண்மையில் என்ன வேலை செய்கிறது என்பதை புரிந்துகொள்கிறவர்கள். அவர்கள் தசைகளை வளர்க்கிறார்கள். இறுதியில் அவர்கள்தான் அறையில் இருக்கும் ஒரே மக்கள்.

இன்று Mac Mini-களிலிருந்து ஏஜென்ட் குழுக்களை இயக்கும் தனி ஆபரேட்டர்கள் எதிர்கால AI செயல்பாடுகளின் தலைவர்கள். அது என்னுக்கு தெரியும் ஏனென்றால் நானே அவர்களில் ஒருவன், இந்த வேலையில் நான் பேசும் மக்களும் அவர்களில் ஒருவர்கள், நாம் யார் என்று நமக்கு தெரியும். க்யூரியாசிட்டி கேட் எங்களுக்காக. ஒரு சுயாதீனரால் கட்டப்பட்டது, சுயாதீனர்களுக்காக, எங்கள் குரலில்.

இதன் கீழ் ஒரு நீண்ட கதை இருக்கிறது, அது என்னவென்றால் ஷார்ட்+ஸ்வீட் 25 ஆண்டுகளாக மனித படைப்பு பங்கேற்பிற்கான அடிப்படை கட்டமைப்பை உருவாக்கி வருகிறது — திருவிழாக்கள், கல்வி, தோன்றி வேலை செய்யும் மக்களுக்கான அங்கீகார அமைப்புகள். ஏஜென்ட் புரட்சிக்கும் அதே வகையான அடிப்படை கட்டமைப்பு தேவை. நிறுவனத்திற்காக அல்ல. உண்மையான வேலை செய்யும் மக்களுக்காக.

க்யூரியாசிட்டி கேட் முதல் துண்டு. தி ஷோ — மனித சிவப்புக்கோடு அனுமதியுடன் ஏஜென்ட் பணி செயல்படுத்தலை ஒருங்கிணைப்பதற்கான கட்டமைப்பு — அடுத்து வருகிறது. தி க்வைன் — ஏஜென்ட்களால் செய்யப்படும் படைப்புப் பணிக்கான நம்பகத்தன்மை அமைப்பு — அதற்கு பிறகு வருகிறது. அனைத்தும் திறந்த மூலம். அனைத்தும் முதலில் சுயாதீன ஆபரேட்டருக்காக கட்டப்பட்டது.

இதில் எதாவது உங்களுக்கு பொருந்தினால், நிறுவுவது ஒரு கட்டளை:

```
npm install curiosity-cat
pip install curiosity-cat
```

தளம் curiositycat.online. மூல குறியீடு GitHub-ல் உள்ளது. வெளியீட்டில் ஐந்து மொழிகள் — ஆங்கிலம், அரபிக், மாண்டரின், இந்தி, தமிழ் — உலகின் AI பயனர்களில் சுமார் 95 சதவீதத்தினரை உள்ளடக்கி.

என்ன உடைகிறது என்பதை கேட்க ஆர்வமாக இருக்கிறேன். மக்கள் தங்கள் கிட்டத்தட்ட-தவறு-ஆனவற்றை அனுப்பினால்தான் டேஞ்சர் மேப் பயனுள்ளதாகும்.
