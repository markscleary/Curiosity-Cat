# 好奇猫（Curiosity Cat）— 产品简介

**每天点击"批准"93次，还是安装好奇猫。**

## 产品定位

好奇猫是一款面向AI智能体的开源安全框架。它介于自主智能体与其所执行任务之间——监测风险、执行常规指令、将险情上报至一张共享的"已发生过什么"危险地图。

现已在npm和PyPI上线。发布时支持五种语言：英语、阿拉伯语、普通话、印地语、泰米尔语。

## 问题所在

在自己设备上运行AI智能体的用户，每天都在不停审批操作。Anthropic数据显示：每位操作者每天平均收到93个审批提示。Gartner预测：到2027年，40%的智能体项目将被取消。Twill、Paperclip等产品都在竞相提升智能体自主性，却无人在构建那一层让操作者决定自己能接受多少风险的控制平面。

## 产品功能

冒险滑块，三个档位：

- **家猫** — 严格隔离，执行常规指令，不越雷池一步
- **街猫** — 接受经过计算的风险，更大胆地探索，但仍会回家
- **猛虎** — 最大活动范围，探索边界，汇报罕见地带与险情故事

操作者自主选择，框架据此调整。敢于冒险的操作者会遭遇他人尚未发现的情况，他们的险情记录将成为危险地图中最有价值的条目。

## 目标用户

在自有硬件上运行智能体的单人操作者和小型团队。独立操作者优先，企业客户在后——这是历史规律：自制电脑俱乐部到苹果，Linux爱好者到全球基础设施，AWS独立开发者到财富500强。

## 安装方式

```
npm install curiosity-cat
pip install curiosity-cat
```

兼容Claude Code、Claude Desktop、Anthropic SDK及自定义智能体。
