# क्यूरियोसिटी कैट — वन-पेजर

**रोज़ 93 बार Approve करें। या क्यूरियोसिटी कैट इंस्टॉल करें।**

## यह क्या है

क्यूरियोसिटी कैट AI एजेंट्स के लिए एक ओपन-सोर्स सुरक्षा फ्रेमवर्क है। यह एक स्वायत्त एजेंट और उसके काम के बीच बैठता है — खतरों पर नज़र रखता है, स्थायी आदेश लागू करता है, और क्लोज़-कॉल रिपोर्टें एक साझा नक्शे पर वापस भेजता है जिसमें यह दर्ज होता है कि बाहर क्या गलत हुआ।

अभी npm और PyPI पर उपलब्ध। लॉन्च पर पाँच भाषाएँ: अंग्रेज़ी, अरबी, मंदारिन, हिंदी, तमिल।

## समस्या

जो लोग अपनी मशीनों पर AI एजेंट्स चलाते हैं वे पूरे दिन कार्रवाइयों को अनुमोदित करते रहते हैं। Anthropic का डेटा: प्रति ऑपरेटर प्रति दिन 93 अनुमोदन संकेत। Gartner: 2027 तक 40% एजेंट प्रोजेक्ट रद्द। Twill, Paperclip और अन्य अधिक स्वायत्तता की ओर दौड़ रहे हैं। कोई भी वह परत नहीं बना रहा जो आपको यह तय करने दे कि आप कितना जोखिम स्वीकार करेंगे।

## उत्पाद

एडवेंचर स्लाइडर (Adventure Slider)। तीन स्थितियाँ:

- **हाउसकैट (घरेलू बिल्ली)** — सख्त क्वारंटाइन, स्थायी आदेश लागू, कुछ भी बाहर नहीं जाता
- **एली कैट (गली की बिल्ली)** — गणना किए गए जोखिम स्वीकार, साहसी अन्वेषण, फिर भी घर वापस आता है
- **टाइगर (बाघ)** — सबसे व्यापक रेंज, किनारे की खोज, खतरे की दुर्लभ जगहों और किस्सों की रिपोर्ट

ऑपरेटर चुनते हैं। फ्रेमवर्क समायोजित होता है। साहसी ऑपरेटर्स वह पाते हैं जो किसी और ने नहीं खोजा। उनके क्लोज़ कॉल डेंजर मैप में सबसे मूल्यवान प्रविष्टियाँ बन जाते हैं।

## किसके लिए है

अकेले ऑपरेटर्स और छोटी टीमें जो अपने हार्डवेयर पर एजेंट्स चलाती हैं। पहले इंडी। एंटरप्राइज़ बाद में — यही ऐतिहासिक पैटर्न है: Homebrew Computer Club से Apple, Linux शौकीनों से वैश्विक इंफ्रास्ट्रक्चर, AWS इंडी डेवलपर्स से Fortune 500।

## कैसे इंस्टॉल करें

```
npm install curiosity-cat
pip install curiosity-cat
```

Claude Code, Claude Desktop, Anthropic SDK, कस्टम एजेंट्स के साथ काम करता है।

## किसने बनाया

शॉर्ट+स्वीट इंटरनेशनल — 50+ शहरों और 15+ देशों में 25 साल की सांस्कृतिक इंफ्रास्ट्रक्चर। क्यूरियोसिटी कैट शॉर्ट+स्वीट एजेंशियल का पहला उत्पाद लॉन्च है, जो कंपनी का AI डिवीज़न है।
