# قطة الفضول — الحقائق الأساسية

**للصحفيين والمحللين. جميع الأرقام موثّقة كما في [DATE].**

## ما هي، في جملة واحدة

إطار أمان مفتوح المصدر لوكلاء الذكاء الاصطناعي يتيح للمشغّلين اختيار مستوى المخاطرة الخاص بهم — من القط المنزلي (صارم) إلى النمر (استكشافي) — ويحوّل تقارير المواقف الخطرة إلى خريطة مشتركة بما جرى بالخارج.

## تفاصيل الإطلاق

- **تاريخ الإصدار:** [DATE]
- **الإصدار عند الإطلاق:** 0.1.1
- **الترخيص:** MIT
- **اسم الحزمة:** `curiosity-cat` (npm + PyPI، كلاهما منشور)
- **التثبيت:** `npm install curiosity-cat` أو `pip install curiosity-cat`
- **الموقع:** curiositycat.online
- **المستودع:** github.com/markscleary/Curiosity-Cat
- **اللغات عند الإطلاق:** الإنجليزية والعربية والصينية والهندية والتاميلية
- **التواصل:** curiosity@shortandsweet.org

## التقنية

- **اللغة:** حزم Python وJavaScript. تعمل مع Claude Code وClaude Desktop وAnthropic SDK وأي إطار عمل يكشف عن خطاف استدعاء الأداة.
- **أوضاع شريط المغامرة:** القط المنزلي، قط الشوارع، النمر
- **مخطط الأوامر الدائمة:** مخطط JSON مفتوح، المساهمات مرحّب بها
- **مخطط خريطة الخطر:** مخطط JSON مفتوح، تقارير المواقف الخطرة موحّدة

## المشكلة التي تحلها

- **93** — متوسط طلبات "موافق" لكل مشغّل يوميًا (بيانات Anthropic المنشورة، الربع الأول من 2026)
- **40%** — توقعات Gartner لمشاريع وكلاء الذكاء الاصطناعي الملغاة بحلول 2027
- **50 مليار دولار** — الإنفاق المؤسسي المقدّر على أدوات وكلاء الذكاء الاصطناعي عام 2025 (IDC)
- **صفر** — منتجات قائمة مصممة لمستوى التحكم في استقلالية المشغّل المنفرد قبل قطة الفضول

## خارطة الطريق

قطة الفضول هي أول منتج يصدر عن شورت آند سويت أجنشيال. الإصدارات التالية في 2026:

- **العرض (The Show)** — إطار تنسيق لتنفيذ مهام الوكلاء مع موافقة بشرية على الخطوط الحمراء. قيد التطوير حاليًا.
- **كوين (The Quine)** — نظام اعتماد للعمل الإبداعي الذي تؤديه الوكلاء.
- **The Green Room** — منصة تبادل عمل للوكلاء ذوي الطاقة الخاملة.
- **Chimera** — إطار تعاون إبداعي متعدد الوكلاء.

## سياق إضافي

تُدير شورت آند سويت إنترناشيونال أكبر دائرة مهرجانات فنون الأداء القصيرة في العالم. تعمل الشركة في الإمارات العربية المتحدة منذ عام 2013، وتُقيم مواسم في دبي وسيدني وملبورن، وتتشارك مع وزارات وجامعات ومؤسسات ثقافية عبر منطقة الخليج وآسيا والمحيط الهادئ. يشغل Mark Cleary منصب الممثل الأسترالي لدى UNESCO في المعهد الدولي للمسرح (ITI).

شورت آند سويت أجنشيال هي قسم الذكاء الاصطناعي في الشركة، تبني أدوات تُطبّق 25 عامًا من الخبرة في البنية التحتية الإبداعية على مشكلات مشغّلي عصر الذكاء الاصطناعي. قطة الفضول هي أول إصداراتها العامة.
