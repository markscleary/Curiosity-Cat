# قطة الفضول — ورقة تعريفية

**انقر "موافق" 93 مرة في اليوم. أو ثبّت قطة الفضول.**

## ما هي

قطة الفضول (Curiosity Cat) إطار أمان مفتوح المصدر لوكلاء الذكاء الاصطناعي. تجلس بين الوكيل المستقل والعمل الذي يؤديه — تراقب بحثًا عن المشكلات، وتُطبّق الأوامر الدائمة، وتُبلّغ عن المواقف الخطرة إلى خريطة مشتركة بما جرى بالخارج.

متاحة الآن على npm وPyPI. خمس لغات عند الإطلاق: الإنجليزية والعربية والصينية والهندية والتاميلية.

## المشكلة

الناس الذين يُشغّلون وكلاء الذكاء الاصطناعي على أجهزتهم الخاصة يقضون أيامهم في اعتماد الإجراءات. بيانات Anthropic: 93 طلب موافقة لكل مشغّل يوميًا. Gartner: 40% من مشاريع الوكلاء ملغاة بحلول 2027. تتسابق Twill وPaperclip وغيرها نحو مزيد من الاستقلالية. لا أحد يبني الطبقة التي تتيح لك تحديد مقدار المخاطرة التي ستقبلها.

## المنتج

شريط المغامرة. ثلاثة أوضاع:

- **القط المنزلي** — حجر صارم، الأوامر الدائمة مُطبَّقة، لا شيء يغادر الساحة
- **قط الشوارع** — تُقبل المخاطر المحسوبة، استكشاف أجرأ، مع العودة للبيت دائمًا
- **النمر** — أوسع نطاق، يستكشف الحافة، يُبلّغ عن أماكن نادرة وحكايات الخطر

المشغّل يختار. الإطار يضبط نفسه. يلتقي المشغّلون الجسورون بما لم يجده أحد غيرهم بعد. مواقفهم الخطرة تصبح أثمن المدخلات في خريطة الخطر.

## لمن هي

المشغّلون المنفردون والفرق الصغيرة التي تُشغّل وكلاء على أجهزتها الخاصة. المستقل أولًا. المؤسسات لاحقًا — هذا هو النمط التاريخي: من نادي Homebrew Computer Club إلى Apple، من هواة Linux إلى البنية التحتية العالمية، من مطوري AWS المستقلين إلى Fortune 500.

## طريقة التثبيت

```
npm install curiosity-cat
pip install curiosity-cat
```

تعمل مع Claude Code وClaude Desktop وAnthropic SDK والوكلاء المخصصة.

## من بناها

شورت آند سويت إنترناشيونال (Short+Sweet International) — 25 عامًا من البنية التحتية الثقافية عبر أكثر من 50 مدينة وأكثر من 15 دولة. قطة الفضول هي أول منتج يصدر عن شورت آند سويت أجنشيال، قسم الذكاء الاصطناعي في الشركة.
