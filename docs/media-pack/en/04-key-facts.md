# Curiosity Cat — Key Facts

**For journalists and analysts. Figures verified against primary sources.**

## One line

An open-source tool that compiles a plain-language risk choice — Housecat, Alley Cat or Tiger — into a real hardened agent permission profile, then proves it with escape trials and a dated Clean Bill: proven, not asserted.

## Release details

- **Current version:** 0.2.x — live on PyPI and npm as `curiosity-cat`
- **First public release:** April 2026 (v0.1); compiler and proof layer: July 2026 (v0.2)
- **Licence:** MIT
- **Install:** `pip install curiosity-cat` or `npm install curiosity-cat`
- **Website:** curiositycat.online · **Repository:** github.com/markscleary/Curiosity-Cat
- **Languages:** English, Arabic, Mandarin, Hindi
- **Contact:** publicist@shortandsweet.org

## Technical

- **Compile:** `curiosity-cat compile --level <housecat|alleycat|tiger> --target claude-code` emits a real Claude Code settings file — allow, deny and ask rules
- **Prove:** `curiosity-cat prove` runs escape trials: self-consistency checks plus, where safe, an observed-deny trial in a live agent session; the two are labelled distinctly on every report
- **Clean Bill:** dated per-trial report (CLEAN-BILL.md) that can fail, and says so
- **Compile targets:** Claude Code today; other frameworks run on standing orders, labelled as guidance
- **Threat taxonomy:** Danger Map anchored to MITRE ATLAS and NIST AI RMF, one canonical schema
- **macOS app:** in final testing — fleet discovery, one-click profile assignment with backup/undo, live guard board
