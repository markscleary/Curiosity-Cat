# Curiosity Cat — Key Facts

**For journalists and analysts. All figures verified as of [DATE].**

## What it is, in one line

An open-source safety framework for AI agents that lets operators choose their own level of risk — from Housecat (strict) to Tiger (exploratory) — and turns close-call reports into a shared map of what's gone wrong out there.

## Launch details

- **Release date:** [DATE]
- **Version at launch:** 0.1.1
- **Licence:** MIT
- **Package name:** `curiosity-cat` (npm + PyPI, both published)
- **Installation:** `npm install curiosity-cat` or `pip install curiosity-cat`
- **Website:** curiositycat.online
- **Repository:** github.com/markscleary/Curiosity-Cat
- **Languages at launch:** English, Arabic, Mandarin, Hindi, Tamil
- **Contact:** curiosity@shortandsweet.org

## Technical

- **Language:** Python and JavaScript packages. Works with Claude Code, Claude Desktop, Anthropic SDK, and any framework that exposes a tool-call hook.
- **Adventure slider positions:** Housecat, Alley Cat, Tiger
- **Standing orders schema:** Open JSON schema, contributions welcome
- **Danger Map schema:** Open JSON schema, close-call reports standardised
