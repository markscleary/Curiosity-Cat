# Curiosity Cat — Key Facts

**For journalists and analysts. All figures verified against primary sources.**

## What it is, in one line

An open-source safety framework for AI agents that shields operators from prompt injection, compromised downloads and deceptive tool endpoints – with an adventure slider from Housecat (strict) to Tiger (exploratory) that turns close-call reports into a shared map of what's gone wrong out there.

## Technical

- **Language:** Python and JavaScript packages. Works with Claude Code, Claude Desktop, Anthropic SDK, and any framework that exposes a tool-call hook.
- **Adventure slider positions:** Housecat, Alley Cat, Tiger
- **Integration:** tool wrappers, file-quarantine watchers, standing orders in the agent system prompt, adapter modules
- **Threat coverage:** prompt injection in fetched content, poisoned tool responses, typosquatted packages, deceptive MCP endpoints, credential-harvesting links, documents with hidden instructions, compromised downloads
- **Standing orders schema:** Open JSON schema, contributions welcome
- **Danger Map schema:** Open JSON schema, close-call reports standardised
