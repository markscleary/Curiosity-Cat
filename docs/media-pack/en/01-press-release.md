### Short+Sweet ships Curiosity Cat v0.2 — agent safety that proves itself

**Sydney —** Short+Sweet AI Labs today released Curiosity Cat v0.2, an open-source safety tool for people who run AI agents without a security team. The new release introduces a compiler and a proof layer: a plain-language risk choice — Housecat, Alley Cat or Tiger — compiles into a real, hardened permission profile for the agent, and the tool then attacks its own walls to prove the profile holds before anyone calls it safe. The result is a dated Clean Bill of health, or an honest list of exactly which wall failed.

"Every security tool asserts. Curiosity Cat demonstrates," said Mark Cleary, founder of Short+Sweet. "You choose how brave your agent is allowed to be. We compile that choice into real allow and deny rules, then we try to break them in a live session and show you what actually got stopped. And where we can't demonstrate, we say so — in writing, on every report."

The honesty is engineered, not promised. Most proof trials are self-consistency checks — the compiled rules replayed against themselves. One, run live when it is safe to do so, is an observed-deny trial: a real agent session asked to do the forbidden thing, and stopped. The two kinds are labelled distinctly on every Clean Bill, and the distinction earned its keep before launch: the proof layer's first observed trial caught a real bug in Curiosity Cat's own compiler — a sandbox setting that quietly bypassed the deny rules. The assertion said safe. The trial said otherwise. The bug was fixed, and the episode became the product's law: proven, not asserted.

The audience is the operator the industry has skipped. Enterprise agent platforms ship with security teams attached; the independent operator running agents on their own machine has had guidance documents and good luck. Anthropic's own documentation notes that Computer Use has no sandbox between the model and what is on the screen — the platform names where its protection ends, and Curiosity Cat begins there. Gartner forecasts that over 40 per cent of agentic AI projects will be cancelled by the end of 2027, citing unclear value and inadequate risk controls, while Deloitte forecasts 75 per cent of companies will invest in agentic AI in 2026. The market is arriving faster than its safety layer.

One compile target ships today: Claude Code, producing a real settings file with allow, deny and ask rules. Every other agent framework runs on Curiosity Cat's standing orders — guidance honestly labelled as guidance, until each target earns its own compiler. A macOS app, bringing discovery, one-click profile assignment and a live guard board to the whole agent fleet, is in final testing.

Curiosity Cat v0.2 is open source under the MIT licence, available now via pip install curiosity-cat and npm install curiosity-cat, in English, Arabic, Mandarin and Hindi. The Danger Map, its shared close-call register anchored to MITRE ATLAS and NIST AI RMF, is live at curiositycat.online.
