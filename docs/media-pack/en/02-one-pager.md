# Curiosity Cat — One-Pager

**A shield for AI agents. Because the agent doesn't know what the world is trying to do to it.**

## What it is

Curiosity Cat is an open-source safety framework for AI agents. It is a shield for independent operators – protecting agents from prompt injection, compromised downloads and deceptive tool endpoints. The operator's work, data, credentials and intellectual property stay theirs.

Available now on npm and PyPI. Five languages at launch: English, Arabic, Mandarin, Hindi, Tamil.

## The problem

An AI agent's safety training protects against the agent doing harmful things. It does not protect the agent from what the world does to it. Prompt injection hidden in documents. Tools that return weaponised outputs. Typosquatted packages. Links that lead to credential harvesting. These threats arrive after training has run. The agent has no immune system.

- Gartner: over 40 per cent of agentic AI projects will be cancelled by the end of 2027, citing unclear value and inadequate risk controls.
- Deloitte: 75 per cent of companies will invest in agentic AI in 2026.

The market is coming. The safety layer is not. Curiosity Cat is the missing piece.

## The product

The Adventure Slider. Three positions:

- **Housecat** – strict quarantine, standing orders enforced, nothing leaves the yard
- **Alley Cat** – calculated risks accepted, braver exploration, still comes home
- **Tiger** – widest range, explores the edge, reports back rare places and tales of danger

Operators choose. The framework adjusts. Brave operators encounter what nobody else has found. Their close calls become the most valuable entries in the Danger Map.

## Who it's for

Solo operators and small teams running agents on their own hardware. Indie first. Enterprise later – that's the historical pattern: Homebrew Computer Club to Apple, Linux hobbyists to global infrastructure, AWS indie devs to Fortune 500.

## How to install

```
npm install curiosity-cat
pip install curiosity-cat
```

Works with Claude Code, Claude Desktop, Anthropic SDK, custom agents.
