# Curiosity Cat — One-Pager

**Tap Approve 93 times a day. Or install Curiosity Cat.**

## What it is

Curiosity Cat is an open-source safety framework for AI agents. It sits between an autonomous agent and the work it's doing — watching for trouble, enforcing standing orders, and reporting close calls back to a shared map of what's gone wrong out there.

Available now on npm and PyPI. Five languages at launch: English, Arabic, Mandarin, Hindi, Tamil.

## The problem

People who run AI agents on their own machines spend their days approving actions. Anthropic's data: 93 approval prompts per operator per day. Gartner: 40% of agent projects cancelled by 2027. Twill, Paperclip and others race toward more autonomy. Nobody is building the layer that lets you decide how much risk you'll accept.

## The product

The Adventure Slider. Three positions:

- **Housecat** — strict quarantine, standing orders enforced, nothing leaves the yard
- **Alley Cat** — calculated risks accepted, braver exploration, still comes home  
- **Tiger** — widest range, explores the edge, reports back rare places and tales of danger

Operators choose. The framework adjusts. Brave operators encounter what nobody else has found. Their close calls become the most valuable entries in the Danger Map.

## Who it's for

Solo operators and small teams running agents on their own hardware. Indie first. Enterprise later — that's the historical pattern: Homebrew Computer Club to Apple, Linux hobbyists to global infrastructure, AWS indie devs to Fortune 500.

## How to install

```
npm install curiosity-cat
pip install curiosity-cat
```

Works with Claude Code, Claude Desktop, Anthropic SDK, custom agents.
