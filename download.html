Curiosity Cat for macOS

Fleet discovery, one-click profile assignment with backup and undo, and a live guard board for every agent on your machine.

In final testing — not yet shipped

Requirements

What it'll need

Planned system requirements for the first release. Subject to change before launch.

  • OS macOS 14 Sonoma or later
  • Chip Apple Silicon (M1 or later) — no Intel build planned for v1
  • Disk ~150 MB free space
  • CLI curiosity-cat installed separately via pip or npm, for compile and prove

Install

How it'll work

Planned install flow — described honestly, ahead of the app existing anywhere but here. Nothing below is available to run yet.

  1. Download the .dmg once it's published on this page, with a SHA-256 checksum alongside it.
  2. Open the .dmg and drag Curiosity Cat to Applications. Standard macOS install, no separate installer.
  3. Launch the app. Gatekeeper will ask you to confirm — the app is planned to be notarised by Apple before release.
  4. Grant discovery permission. The app scans for agent workspaces and existing curiosity-cat profiles on your Mac; nothing leaves the machine.
  5. Assign a profile to each agent from the guard board, with one-click backup and undo before anything is applied.
