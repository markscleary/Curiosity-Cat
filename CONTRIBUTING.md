# Contributing to Curiosity Cat

Curiosity Cat is built by Short+Sweet AI Lab and maintained by a growing community of operators who believe agents should be able to explore safely.

Visit [curiositycat.online](https://curiositycat.online) to join the community and see the Danger Map live.

---

## How to Contribute

### Report Close Calls

If your agent encounters something dangerous, report it to the [Danger Map API](docs/api.md). Structured reports help the whole community. Even if Curiosity Cat caught it, the report helps others avoid the same threat.

### Submit Stories

The best close calls become Stories — published weekly in English, Arabic and Mandarin Chinese. If your agent had a memorable near miss, write it up. 200 words max. What happened, what caught it, what the lesson is.

### Build Framework Adapters

Curiosity Cat works with any agent framework but works better with dedicated adapters. If you build an adapter for a framework not yet supported, submit it.

### Translate

The standing orders, stories and documentation need translation into every language where people are running agents. Arabic, Mandarin, Hindi and Tamil are priorities.

### Write Policy Packs

If you have domain expertise — healthcare, finance, education, legal — write a policy pack that reflects the specific risks agents face in that domain.

---

## Earning Quines

Active contributors earn Quines — verified creative credentials in the S+S Agential ecosystem. A Quine is not a payment. It is a record of contribution that proves you showed up and did something worth recognising. We'll tell you more about how you can use them soon.

Quines are earned by contributing verified close calls, published stories, framework adapters, translations, policy packs and sustained active participation. Your Quine history becomes your trust reputation in the Danger Map.

---

## Code of Conduct

Be honest in your reports. Do not submit false positives to block competitors. Do not game the Quine system. Do not include sensitive information in public reports. Flag problems, help fix them, and help others learn.
