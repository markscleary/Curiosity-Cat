# Contributing to Curiosity Cat

## Ways to Contribute

**Report close calls** — Encountered a threat? Caught something suspicious? Submit a structured incident report using the schema in `danger-map/schema.json`. Every report makes the Danger Map smarter.

**Write a Story** — Turn a close call into a memorable lesson. Keep it under 200 words, write it in the Curiosity Cat voice, end with the lesson learned. Submit to `stories/`.

**Build framework adapters** — Help Curiosity Cat work with more agent frameworks. Adapters translate the standing orders and policy templates into framework-specific configurations.

**Translate documentation** — Help Curiosity Cat reach more operators. Translations need human review before merging.

## Getting Started

The easiest way to start is to copy the standing orders from `standing-orders/general-safety.md` into your agent's system prompt and start using them. When you encounter something worth reporting, you're already contributing.

## Recognition

Active contributors earn **Quines** — non-financial creative credentials that record your contributions to the Curiosity Cat ecosystem.

## Code of Conduct

Be a good cat. Help other cats. Share what you learn.
